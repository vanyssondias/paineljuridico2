# Painel Jurídico — Gestão de Agenda, Processos e Contratos

Painel web para advogados, em página única (HTML/CSS/JS), sem backend ou banco de dados. Os dados ficam salvos no navegador (localStorage).

## Funcionalidades
- Agenda com calendário visual e linha do tempo
- Gestão de processos com prazos e fases
- Tarefas e checklist de legislações
- Gestão de contratos com checklist de análise
- Painel de prazos críticos (3/7/30 dias)

## Como usar
Acesse a página publicada pelo GitHub Pages (link nas Settings do repositório) ou abra o arquivo `index.html` localmente em qualquer navegador.

## Tecnologia
HTML, CSS e JavaScript puro. Sem dependências externas, sem necessidade de servidor.

## Privacidade
Todos os dados são salvos apenas no navegador do usuário (localStorage). Nada é enviado para servidores externos.
